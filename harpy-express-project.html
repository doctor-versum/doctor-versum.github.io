Direkte Integration von Modpacks, Mods und Shadern.

1

Prism Launcher installieren

Lade dir die neueste Version von der offiziellen Website herunter. Prism ist für Windows, macOS und Linux verfügbar.

Download Prism Launcher
Wichtig: Stelle sicher, dass du die passende Java-Version (mindestens Java 21 für das Modpack) installiert hast!
Download Java 25 von openJDK
2

Minecraft Account hinzufügen

Nach dem ersten Start musst du deinen Account verknüpfen:

  • Klicke oben rechts auf "Profiles".
  • Wähle "Manage Accounts".
  • Klicke auf "Add Microsoft" und folge den Anweisungen im Browser.
3

Modpack von Modrinth finden

Du musst die Modrinth-Website nicht einmal verlassen, um Packs zu finden – Prism erledigt das für dich:

  • Klicke oben links auf "Add Instance" (Instanz hinzufügen).
  • Wähle im linken Menü den Reiter "Modrinth".
  • Suche oben nach "The last voyage of the Harpy express".
  • Wähle das Modpack mit dem gold-braun belichteten zug aus.
  • Wähle die oberste Version im dropdown aus und klicke auf "OK".
Tipp: Du kannst auch eine .mrpack Datei von Modrinth herunterladen und sie einfach per Drag & Drop in das Prism-Fenster ziehen!
Download "The last voyage of the Harpy Express" auf Modrinth Direkter download der .mrpack
4

Starten & Genießen

Doppelklicke auf das neue Icon deiner Instanz. Der Launcher lädt nun alle benötigten Dateien und Mods automatisch herunter.
