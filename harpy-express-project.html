1

Prism Launcher installieren

Lade dir die neueste Version von der offiziellen Website herunter. Prism ist für Windows, macOS und Linux verfügbar.

Download Prism Launcher
Wichtig: Installiere Java 21 (empfohlen), damit das Pack stabil läuft.
Download Java 21 (Temurin)
2

Minecraft Account hinzufügen

Nach dem ersten Start musst du deinen Account verknüpfen:

  • Klicke oben rechts auf "Profiles".
  • Wähle "Manage Accounts".
  • Klicke auf "Add Microsoft" und folge den Anweisungen im Browser.
3

Modpack finden & installieren

Suche direkt in Prism nach unserem Pack:

  • Klicke auf "Add Instance" -> Reiter "Modrinth".
  • Suche nach The last voyage of the Harpy express.
  • Wähle das Pack mit dem gold-braun belichteten Zug aus.
  • Wähle die oberste Version und klicke auf "OK".
Tipp: Manueller Weg über die Website:
Modpack auf Modrinth Direkt-Download .mrpack
4

Voicechat & Mikrofon

So stellst du dein Mikrofon im Spiel ein:

  • Drücke im Spiel "V" für das Voicechat-Menü.
  • Klicke auf "Settings".
  • Wähle unter "Select Microphone" dein Gerät aus.
  • Teste es mit "Enable Microphone Test".
Empfehlung: Headset verwenden, um Echos für die anderen Spieler zu vermeiden!
5

Vorab-Erkundung

Du kannst den Zug bereits im Einzelspieler besuchen, um sicherzugehen, dass dein PC das Pack flüssig darstellt.
